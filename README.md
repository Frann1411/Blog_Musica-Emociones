# Música y Emociones

Sitio web estático en español dedicado a la relación entre la música y las emociones. El proyecto reúne páginas sobre artistas, canciones, playlists, una sección personal y un teaser de un próximo proyecto inspirado en Gustavo Cerati.

## Descripción

**Música y Emociones** es un blog visual con una identidad oscura, atmosférica y emocional. La idea del sitio es explorar cómo la música conecta con recuerdos, estados de ánimo y experiencias personales, combinando contenido narrativo con una presentación cuidada para escritorio y móvil.

## Secciones del sitio

- **Inicio**: presentación principal del blog y contenido destacado.
- **Artistas**: recorrido por artistas influyentes y su impacto emocional.
- **Canciones**: colección de canciones con descripciones personales.
- **Playlists**: listas temáticas para distintos momentos.
- **Sobre mí**: contexto personal del proyecto y su enfoque.
- **Universo Cerati**: página teaser para un próximo proyecto inmersivo.

## Tecnologías

- HTML5
- CSS3
- Tailwind CSS
- JavaScript
- Netlify para despliegue
